Requirements for the API

The rbxdatabase API is very simple to use. A couple things you need to know is that the API can only be used in online servers from server-side scripts. This is a requirement of the roblox ModuleScripts.

Setting up the API

OK, first thing you need to do is get the API by requiring the module, which can be accomplished like so:

local RBXDatabase = require(167431702) -- Asset id of the rbxdatabase module.

This defines the API as RBXDatabase in the script. This means we just need to access that variable when we want to use the api. You can define the API as anything you like, you don't have to define it as RBXDatabase.

Connecting to the Website

Now that we have the API defined, we need to connect using our API key. Each script needs to have it's own connection to the API, so each script needs to call the Connect method.

RBXDatabase:Connect("PUT YOUR API KEY HERE") -- Your api key is used for connecting. Pass it as a string to the Connect method.

That's it! You connected to the website. Now you need to save a value.

Selecting a Data Group

The API has a feature to "separate" data into groups. The SelectDataGroup method will pick the group.

RBXDatabase:SelectDataGroup("My Data Group") -- Now all API actions will be made to this data group.

If you don't use SelectDataGroup, it will default to a blank string("").

Saving a Value

OK, you connected. Now what? Now you need to learn to save a value by it's key. Rbxdatabase can store pretty much any data type, including roblox-specific ones. You can store values by using the SetValue method.

RBXDatabase:SetValue("KeyName", "ValueHere") -- Saves the key "KeyName" with the string value "ValueHere".

You just saved a value! Not hard, is it?

Getting a Value

Now that you stored a value, you probably want to get that value back as well. That is what the GetValue method does.

print(RBXDatabase:GetValue("KeyName")) -- Prints the value of the key "KeyName" into the output, or nil if the key doesn't exist.

Basic enough, right?

Deleting a Key

What if you don't want to use a key anymore? Delete it using the DeleteValue method.

RBXDatabase:DeleteValue("KeyName") -- Deletes the key "KeyName".
print(RBXDatabase:GetValue("KeyName")) -- Prints nil into the output because the key doesn't exist anymore.

Updating a Value

If you want to take the old value of a key and append to it, use the UpdateValue method.

RBXDatabase:UpdateValue("KeyName",function(OldValue) return OldValue+1 end) -- Assuming the value of "KeyName" is a number, it takes the old value, adds one, and sets it to that.

UpdateValue takes two variables. The key of the value you want to edit and a function to return the new value of the key.

Getting all of Your Data

What if you want to get all of your data from the current data group into one big table? Use the GetAllData method.

local Data = RBXDatabase:GetAllData()

for i, v in pairs(Data) do -- Loops through the return table and prints each of your values and their corrosponding keys.
     print(i, v)
end

The return value will be a table of all of your data in the specified data group. Each value is indexed in the table by it's key.

Disconnecting from the Website

If you want to close the connection to the website, use Disonnect.

RBXDatabase:Disconnect() -- This closes the connection to the website.

When the Disconnect method is called the connection closed, and all functions but Connect will stop working.
